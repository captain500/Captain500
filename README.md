### 🚀 Featured Project: Canada Rental Market Intelligence
*An end-to-end Modern Data Stack platform to analyze the Canadian real estate market.*

This project implements a **Medallion Architecture** to provide insights into Canadian rental trends.

* **Ingestion:** Scrapy-based crawlers.
* **Storage:** **MinIO** for the Bronze layer.
* **Processing:** **Polars** for fast cleaning and **PostgreSQL** as the analytical engine.
* **Transformation:** **dbt** implementing a **Kimball Star Schema** (Fact & Dimension tables).
* **Quality & DevOps:** **Great Expectations** for schema validation, **Poetry** for dependency management, and **GitHub Actions** for CI/CD.
* **Orchestration:** **Prefect** managing complex flows with automated retries and logging.
* **Visualization:** Interactive **Power BI** dashboards.
